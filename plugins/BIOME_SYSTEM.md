# Dungeon Biome & Magic Beast System

## Overview
Dungeons are now **themed by biome**, with each biome spawning specific magic beast families. This creates lore-accurate, immersive dungeon experiences where the environment matches the monsters you encounter.

---

## Biome Classification System

### **9 Unique Biomes:**

Each biome has:
- Unique name pool for dungeons
- Specific magic beast families
- Themed boss names
- Mob count multiplier
- HP scaling for bosses
